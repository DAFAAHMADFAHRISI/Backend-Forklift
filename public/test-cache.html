Test Cache API (5 detik)

Test Cache Data Statis

Data ini akan tetap sama selama 5 detik karena di-cache

Test Cache Data Dinamis

Data ini akan berubah setiap request, tapi di-cache selama 5 detik

Instruksi Testing:

  1. Klik tombol "Test Cache Statis" beberapa kali - data akan tetap sama selama 5 detik
  2. Klik tombol "Test Cache Dinamis" beberapa kali - data akan tetap sama selama 5 detik
  3. Tunggu 5 detik, lalu klik lagi - data akan diperbarui
  4. Perhatikan timestamp untuk melihat kapan data terakhir diperbarui
  5. Timer akan menunjukkan berapa detik tersisa sebelum cache expired
